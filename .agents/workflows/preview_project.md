# Preview Project Workflow

This workflow details the step-by-step commands to start the local development server and test the web application.

## Prerequisites

- Node.js and dependencies installed (`npm install` has been run)

## Step 1: Start the Development Server

To start the local Next.js development server with Turbopack, run:

```bash
npm run dev
```

*Note: By default, the development server is configured to run on port `9002` in `package.json`.*

## Step 2: Access the Application

Once the server is running, open your web browser and navigate to:

- Login page: [http://localhost:9002/login](http://localhost:9002/login)
- Home / Dashboard: [http://localhost:9002/](http://localhost:9002/)
