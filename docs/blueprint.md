# **App Name**: MA WORK JP Portal

## Core Features:

- Company Management: Register, edit, and delete corporate clients, managing their details and contract plans.
- Employee Management: Register, edit, and delete employee information, including personal details, visa status, and employment dates.
- Visa and Contract Alerting: Set up and manage automated alerts for upcoming visa and contract expiration dates, notifying MA WORK JP, company representatives, and legal professionals.
- Legal Professional Management: Manage contact information for legal professionals associated with each company.
- Email Notification System: Send customized email notifications with templated messages to relevant parties regarding visa and contract expirations.
- Document Generation: Generate and download PDF reports for employee lists, visa expirations, contract renewals, and monthly reports, including company logos.
- Dashboard Overview: Display key metrics on the dashboard, such as the number of client companies, total foreign employees, upcoming renewals, and alerts.

## Style Guidelines:

- Primary color: A professional deep blue (#1A3A7B) conveying trust and stability.
- Background color: A very light desaturated blue (#F0F4FA) providing a clean and spacious feel.
- Accent color: A slightly muted light blue (#5C85D3) for interactive elements and highlights, maintaining a consistent yet distinct look.
- Primary font: 'Inter' (sans-serif) for clean, legible headlines and body text, ensuring a modern and objective feel suitable for a management portal.
- Code font: 'Source Code Pro' for any technical readouts or data logs.
- Use minimalist, solid icons that clearly represent functions without distraction, fitting the professional management portal aesthetic.
- Implement a responsive dashboard layout that adapts seamlessly between desktop and mobile devices, prioritizing clarity and ease of navigation.
- Apply subtle fade-in animations for data loading and transition effects to enhance user experience without being obtrusive.
